## 仕様・注意
- **形式**：JPEG（sRGB）。1枚あたり **500KB 以下**を目安に圧縮（表示速度のため）。
- **著作権**：対外レポートのため、**商用利用可・帰属不要**のフリー素材（Unsplash / Pexels 等）または自前撮影を使用。特定できる人物の顔・他社ロゴの写り込みは避ける。
- **クレジット**：帰属が必要な素材を使う場合は、撮影者名をここに追記 → フッターに記載します。
